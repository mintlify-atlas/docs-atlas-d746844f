## Overview

The banner system is Nozmie's primary UI for presenting teleport and utility options. It features an elegant, interactive design with support for multiple options, cooldown tracking, and intelligent stacking.

## Banner Stacking

### How Stacking Works

When a new banner appears while one is already visible, it stacks above with smooth animations:

```lua
function BannerController.ShowWithOptions(banner, teleportOptions, isStacked, allowStack)
    if not isStacked and banner:IsShown() and allowStack then
        banner.stack = banner.stack or {}
        local stackedBanner = BannerUI.CreateBanner()
        stackedBanner.stackRoot = banner
        table.insert(banner.stack, stackedBanner)
        
        local anchor = GetStackAnchor(banner, #banner.stack)
        ApplyAnchor(stackedBanner, anchor)
    end
end
```

### Stack Layout

From `BannerController.lua:23`:

```lua
local STACK_GAP = 8

local function GetStackAnchor(root, index)
    local base = GetBaseAnchor(root)
    return {
        point = base.point,
        relativeTo = base.relativeTo,
        relativePoint = base.relativePoint,
        xOfs = base.xOfs,
        yOfs = base.yOfs + (Config.BANNER.HEIGHT + STACK_GAP) * index
    }
end
```

<Info>
  Each stacked banner is offset by **76 pixels** (68px height + 8px gap) from the previous one.
</Info>

### Stack Promotion

When the main banner is closed, the next stacked banner slides down to take its place:

```lua
local function PromoteNextBanner(root)
    if not root.stack or #root.stack == 0 then
        return
    end
    
    local nextBanner = table.remove(root.stack, 1)
    local baseAnchor = GetBaseAnchor(root)
    
    SlideToAnchor(nextBanner, baseAnchor, 0.2, function()
        root.options = nextBanner.options
        RefreshBannerDisplay(root)
        ReflowStack(root)
    end)
end
```

<Accordion title="Animation Details">
  - **Duration**: 0.2 seconds
  - **Type**: Translation animation
  - **Effect**: Smooth slide to new position
  - **Post-animation**: Content refresh and stack reflow
</Accordion>

### Cooldown State

From `BannerController.lua:223`:

```lua
local function UpdateBannerForCooldown(banner, data, remaining)
    IconHandling.SetDesaturation(banner.icon, true)
    banner.title:SetTextColor(unpack(Config.COLORS.TEXT_COOLDOWN))
    banner:SetBackdropColor(unpack(Config.COLORS.BACKDROP_COOLDOWN))
    banner.isOnCooldown = true
    
    local cooldownTitle = string.format("%s is on cooldown", GetEntryName(data))
    banner.title:SetText(cooldownTitle)
    
    local timeText = Helpers.FormatCooldownTime(remaining)
    if Helpers.IsInAnyGroup() then
        banner.subtitle:SetText("|cffff0000" .. timeText .. "|r |cff888888 - Click to announce|r")
    else
        banner.subtitle:SetText("|cffff0000" .. timeText .. "|r")
    end
end
```

<Warning>
  When a spell is on cooldown, clicking the banner will announce the cooldown to your group instead of casting.
</Warning>

## Cooldown Tracking

Banners automatically update every 0.5 seconds to track cooldowns:

```lua
banner.updateTimer = C_Timer.NewTicker(0.5, function()
    RefreshBannerDisplay(banner)
```

## Combat Lockdown

### State Management

From `BannerController.lua:28`:

```lua
local function ApplyBannerCombatState(banner)
    if IsCombatLocked() then
        SetBannerInteractiveState(banner, false)
        banner:SetAlpha(0.2)
        banner.nozmieCombatDimmed = true
    else
        SetBannerInteractiveState(banner, true)
        if banner.nozmieCombatDimmed then
            banner:SetAlpha(1)
            banner.nozmieCombatDimmed = false
        end
    end
end
```

## Duplicate Detection

Nozmie prevents duplicate banners from appearing:

```lua
function BannerController.FindBannerByOptions(root, options)
    local optionsKey = BuildOptionsKey(options)
    if GetBannerOptionsKey(root) == optionsKey then
        return root
    end
    if root.stack then
        for _, frame in ipairs(root.stack) do
            if GetBannerOptionsKey(frame) == optionsKey then
                return frame
            end
        end
    end
    return nil
end
```

From `Init.lua:119`:

```lua
local existingBanner = BannerController.FindBannerByOptions(banner, matches)
if existingBanner then
    -- Refresh existing banner instead of creating new one
    BannerController.ShowWithOptions(existingBanner, matches, isStacked, false)
    return
end
```

<Note>
  If you receive the same request multiple times, Nozmie will refresh the existing banner instead of stacking duplicates.
</Note>
