## Overview

Nozmie provides two primary slash command aliases and several subcommands for quick access to features and settings.

<CodeGroup>
```bash Primary
/nozmie
```

```bash Short alias
/noz
```

```bash Utility UI
/nozui
```
</CodeGroup>

## Command Registration

From `Init.lua:195`:

```lua
SLASH_NOZMIE1 = "/nozmie"
SLASH_NOZMIE2 = "/noz"
SlashCmdList["NOZMIE"] = HandleCommand

SLASH_NOZUI1 = "/nozui"
SlashCmdList["NOZUI"] = function()
    UtilityUI.Show()
end
```

<Tabs>

  </Tab>
  
  <Tab title="Implementation">
    From `Init.lua:137`:
    ```lua

        local words = args:match("^blacklist%s+(.+)")
        Settings.Set("blacklistedWords", words)
        print("|cff00ff00Nozmie:|r Blacklist updated to: |cffFFFFFF" .. 
              words .. "|r")
    end
    ```
  </Tab>
  
  <Tab title="Output">
    ```
    Nozmie: Blacklist updated to: summon, lfg, boost
    ```

  </Tab>

<Note>
  Blacklisted words are comma-separated. Messages containing any blacklisted word will be completely ignored by the detection system.
</Note>

## Last Banner

### Show Last Banner

<Tabs>
  <Tab title="Command">
    ```bash
    /noz last
    ```
  </Tab>
  
  <Tab title="Implementation">
    From `Init.lua:162`:
    ```lua
    elseif cmd == "last" then
        _G.Nozmie_ShowLastBanner()
    end
    
    -- From Init.lua:53
    _G.Nozmie_ShowLastBanner = function()
        local last = BannerController.GetLastOptions()
        if last and #last > 0 then
            _G.Nozmie_ShowOptions(last)
        else
            print("|cff00ff00Nozmie:|r No recent banner to show.")
        end
    end
    ```
  </Tab>
  
  <Tab title="Result">
    - Shows the most recently displayed banner
    - Useful if you closed a banner by accident
    - Can also be triggered by clicking the minimap icon
  </Tab>
</Tabs>

<Warning>
  The last banner is only stored in memory during your current session. It's cleared when you reload the UI or log out.
</Warning>

```lua

Nozmie_UtilityUI.Hide()
```

From `UtilityUI.lua:241`:
```lua
function UtilityUI.Toggle()
    local host = EnsureFrame()
    if host:IsShown() then
        host:Hide()
    else
        host:Show()
    end
end
```

## Command Handler Flow

From `Init.lua:132`:

```lua
local function HandleCommand(args)
    local cmd = args:lower():trim()
    
    -- Settings commands
    if cmd == "settings" or cmd == "config" or 
       cmd == "options" or cmd == "" then
        Settings.Show()
    
    -- Blacklist commands
    elseif cmd:match("^blacklist%s+(.+)") then
        -- Set blacklist
    elseif cmd == "blacklist" then
        -- View blacklist
    
    -- Minimap toggle
    elseif cmd == "minimap" or cmd == "mm" then
        -- Toggle minimap icon
    
    -- Last banner
    elseif cmd == "last" then
        -- Show last banner
    
    -- Help (default)
    else
        -- Display help text
    end
end
```
