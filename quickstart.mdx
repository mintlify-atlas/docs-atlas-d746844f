## Your First Steps with Nozmie

This guide will walk you through using Nozmie for the first time, from basic setup to casting your first spell.

<Note>
  Before starting, make sure you've [installed Nozmie](/installation) and can see the minimap icon in-game.
</Note>

## Step 1: Open the Settings

First, let's familiarize ourselves with the Nozmie interface.

<Steps>
  <Step title="Access Settings">
    Type `/noz` or `/nozmie` in chat to open the settings panel.
    
    Alternatively, click the Nozmie minimap icon.
  </Step>
  
  <Step title="Verify Addon is Enabled">
    In the settings panel, ensure the "Enable Nozmie" checkbox is checked.
    
    This is the master toggle for all addon functionality.
  </Step>
  
  <Step title="Configure Chat Channels">
    Select which chat channels Nozmie should monitor:
    - **Say** - Local /say messages
    - **Party** - Party chat
    - **Raid** - Raid chat  
    - **Guild** - Guild chat
    - **Whisper** - Direct whispers
    - **Battle.net Whisper** - Cross-realm whispers
    
    <Info>
      Start by enabling **Party** and **Raid** channels for group content.
    </Info>
  </Step>
</Steps>

## Step 2: Test Chat Detection

Now let's see Nozmie in action by testing the chat detection system.

### Example 1: Teleportation Request

If you're a class with teleportation abilities (Mage, Druid, Monk, Death Knight, etc.):

<Steps>
  <Step title="Type a Test Message">
    In /say or /party chat, type a message containing a dungeon or location name:
    
    ```
    Can someone port to Dornogal?
    ```
    
    ```  
    Need teleport to Ara-Kara
    ```
  </Step>
  
  <Step title="Watch for the Banner">
    A banner will appear on your screen showing available teleport options.
    
    The banner displays:
    - Spell/item icon
    - Destination name
    - Category (M+ Dungeon, Raid, etc.)
    - Cooldown status
  </Step>
  
  <Step title="Click to Cast">
    Click the banner to instantly cast the spell or use the item.
    
    Nozmie will automatically target the requesting player if applicable.
  </Step>
</Steps>

<Note>
  The banner auto-hides after a configurable timeout (default: 10 seconds). You can adjust this in settings or recall the last banner with `/noz last`.
</Note>

### Example 2: Utility Request

If you can provide buffs or utilities (any class):

<CodeGroup>
```text Mage Examples
Can I get food?
Need int buff
Can someone slowfall me?
```

```text Warlock Examples

```

```text Priest Examples

```
</CodeGroup>

When you or a party member types these messages, Nozmie displays the appropriate utility spells you can cast.

## Step 3: Use the Utility Page

The Utility Page is a browsable interface for all your available teleports, portals, and utilities.

<Steps>
  <Step title="Open Utility Page">
    Type `/noz utility` in chat to open the Utility Page.
  </Step>
  
  <Step title="Browse Tabs">
    Explore the organized tabs:
    
    - **Current Dungeons** - M+ dungeons in the current season (priority teleports)
    - **Legacy Dungeons** - Previous season M+ dungeons  
    - **Teleports** - Raid teleports, Delve access, and teleportation toys
    - **Utility** - Class utilities, buffs, summons, repair bots, etc.
    - **Hearthstone** - All your hearthstone items and variants
  </Step>
  
  <Step title="Use the Search Bar">
    Click the search bar and type a location or spell name:
    
    ```
    dalaran
    ```
    
    The grid filters in real-time to show matching items across all tabs.
  </Step>
  
  <Step title="Click to Cast">
    Click any icon to instantly cast that spell or use that item.
    
    Icons display cooldown timers when abilities are on cooldown.
  </Step>
</Steps>

<Info>
  The Utility Page only shows items and spells you actually own and can use - keeping your interface clean and relevant.
</Info>

## Step 4: Customize Your Experience

Tailor Nozmie to your playstyle with these essential settings.

### Configure Suppression Filters

Control what appears in your banners:

<Steps>
  <Step title="Open Settings">
    Type `/noz` to open the settings panel.
  </Step>
  
  <Step title="Set Global Filters">
    Under "Global Suppression," select categories to hide everywhere:
    - **Mounts** - Hide mount-related utilities
    - **Class Utilities** - Suppress class buffs and services
    - **Teleports & Portals** - Filter teleportation options
    - **Utility Services** - Hide repair, mailbox, transmog, anvil
  </Step>
  
  <Step title="Set Instance Filters">
    Under "Instance Suppression," select categories to hide only in dungeons and raids.
    
    <Info>
      **Example:** Suppress mounts in instances but show them in the open world by adding "Mounts" to instance filters only.
    </Info>
  </Step>
</Steps>

### Customize Banner Behavior

<Accordion title="Banner Timeout">
  Set how long the banner remains visible before auto-hiding:
  
  - Default: 10 seconds
  - Range: 5-60 seconds
  - Set to 0 for no auto-hide (manual close only)
</Accordion>

<Accordion title="Portal Priority">
  When both portal and teleport spells match, prioritize showing portals:
  
  ```lua
  Enable "Prefer Portals" in settings
  ```
  
  This is useful for Mages who want to offer portals to party members rather than personal teleports.
</Accordion>

<Accordion title="Group Announcements">
  Automatically announce to your group when you cast a spell from a banner:
  
  - Announces to party/raid chat
  - Includes spell name and target player
  - Can be disabled in settings
</Accordion>

### Blacklist Noisy Keywords

Prevent false positives by blacklisting words:

```lua
/noz blacklist portable, important, report
```

This prevents Nozmie from triggering when these words appear in messages.

<Note>
  View your current blacklist with `/noz blacklist` (no arguments).
</Note>

## Combat Behavior

<Info>
  When you enter combat, Nozmie banners become semi-transparent and non-interactive to prevent accidental clicks and interface taint.
  
  Any matches detected during combat are queued and displayed automatically when you leave combat.
</Info>

## Tips for Success

<AccordionGroup>
  <Accordion title="Keyword Matching is Intelligent">
    Nozmie uses whole-word pattern matching:
    - ✅ "need **portal**" → Matches portal spells
    - ❌ "this is **portable**" → Does NOT match (prevents false positives)
    - ✅ "tele to **orgrimmar**" → Matches Orgrimmar teleport
  </Accordion>
  
  <Accordion title="Stacked Banners for Multiple Matches">
    When multiple items match a single message, Nozmie displays all options in stacked banners.
    
    Navigate using arrow buttons on the banner or click the specific banner you want.
  </Accordion>
  
  <Accordion title="Use Utility Page for Quick Access">
    Bind `/noz utility` to a hotkey for instant access to all your utilities without waiting for chat triggers.
    
    Great for raid leaders or players who frequently provide utilities.
  </Accordion>
  
  <Accordion title="Customize Per Character">
    Nozmie settings are saved per-character, allowing different configurations:
    - Enable all channels on support characters (Mage, Warlock)
    - Minimal monitoring on DPS-only characters
    - Different suppression filters for different roles
  </Accordion>
</AccordionGroup>
